# Shufti Pro

Shufti Pro is an AI-powered identity verification platform offering a REST API for document verification, face biometrics, AML screening, business verification (KYB), and address verification. The platform supports 10,000+ document types across 230+ countries and territories, integrates with 1700+ AML watchlists, and provides mobile SDKs for Android, iOS, Flutter, and React Native.

**Website:** [https://shuftipro.com/](https://shuftipro.com/)
**Developer Docs:** [https://developers.shuftipro.com/](https://developers.shuftipro.com/)
**GitHub:** [https://github.com/shuftipro](https://github.com/shuftipro)

## APIs

- **Verification API** — Core identity verification: documents, facial biometrics, address, AML, phone/email MFA, VideoIdent, e-signatures
- **KYB API** — Business verification from official registries and business AML screening
- **AML Screening API** — Screening against 1700+ global watchlists, sanctions lists, and PEP databases
- **Biometric Authentication API** — Face-based re-authentication with liveness detection for returning users

## Authentication

REST API using HTTP Basic Auth (Client ID as username, Secret Key as password). Responses validated via SHA256 signature verification.

## SDKs

Android, iOS, Flutter, and React Native mobile SDKs available at [https://github.com/shuftipro](https://github.com/shuftipro).
